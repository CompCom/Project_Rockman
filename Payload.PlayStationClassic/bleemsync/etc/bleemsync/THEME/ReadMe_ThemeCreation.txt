- Avoid deleting neither the "modmyclassic" or "Stock" folders.
- When creating themes, DUPLICATE the "modmyclassic" folder and edit the assets accordingly.
- If a file is misnamed, it will either not be desplayed OR assets from "Stock" will be displayed in it's stead.
- Looking where to drop folder images? [USB-root]\bleemsync\etc\bleemsync\SUP\launchers\folder_launch\images
- Looking where to change the Launcher icons on the main carousel? [USB-root]\bleemsync\etc\bleemsync\SUP\launchers\
